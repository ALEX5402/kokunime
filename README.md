<h2>
    Setup Local
</h2>

```bash
# cloning this project
git clone https://github.com/kochan4php/kokunime.git

# go to the project folder
cd kokunime

# install pnpm if you does'nt have
npm i -g pnpm

# install dependencies
pnpm install

# run the project
pnpm dev
```

</h2>

```bash

# go to the project folder

sh docker.sh
```
